# Kine App

Application de suivi de rééducation permettant à des kinésithérapeutes de créer et suivre des programmes d'exercices pour leurs patients.

## Stack technique

### Backend

* Symfony 7
* PHP 8.3
* Doctrine ORM
* PostgreSQL

### Infrastructure

* Docker
* Docker Compose
* Nginx

## Lancement du projet

```bash
docker compose up -d
```

Application Symfony :

http://localhost:8080
